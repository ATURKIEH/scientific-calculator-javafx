# Scientific Calculator (JavaFX)

A desktop scientific calculator built using **JavaFX** with a custom
expression evaluation engine.

## Features
- Arithmetic operations
- Parentheses
- Trigonometric functions (sin, cos, tan)
- Logarithms
- Power operator
- Clean JavaFX UI

## Tech Stack
- Java 17
- JavaFX
- Maven (multi-module project)

## Structure
- `engine/` – expression parsing & evaluation
- `ui/` – JavaFX user interface
